.. note:: Some computer security suites in their default configurations can make
   the Denodo Platform function incorrectly. If after the installation you
   cannot create new browsers from the ITPilot Wrapper Generation tool,
   create browser instances at runtime, or use the MSIE ITPilot toolbar,
   check that your security software is not blocking any Denodo application
   or any port used by them.
