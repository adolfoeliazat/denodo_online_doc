This section describes the syntax of condition functions (that can also
be used to define derived fields). These functions can be grouped
into different types based on the data type to which they are applied:

-  Arithmetic functions: see section :doc:`Arithmetic Functions </vdp/vql/appendix/syntax_of_condition_functions/arithmetic_functions>`.
-  Text processing functions: see section :doc:`Text processing Functions </vdp/vql/appendix/syntax_of_condition_functions/text_processing_functions>`.
-  Date processing functions: see section :doc:`Date Processing Functions </vdp/vql/appendix/syntax_of_condition_functions/date_processing_functions>`.
-  XML processing functions: see section :ref:`XML Processing Functions`.
-  Type conversion functions: see section :doc:`Type Conversion Functions </vdp/vql/appendix/syntax_of_condition_functions/type_conversion_functions>`.
-  Aggregation functions: see section :ref:`Aggregation Functions`.
-  Analytic functions: see section :ref:`Analytic Functions (Window
   Functions)`.
-  Other functions: see section :ref:`Other Functions`.
