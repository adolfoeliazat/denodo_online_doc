=====================
Creating Data Sources
=====================


In Virtual DataPort, wrappers are associated to data sources. Data
sources encapsulate the configuration needed to access a certain data
repository (e.g. a database, a Web service, etc.)

The following sections describe the manual creation process for each
data source type:
