========================================================
Configuration of the ODBC Driver in Linux and Other UNIX
========================================================

.. toctree::
   :hidden:

   set_up_a_dsn_on_linux_and_other_unix
   compile_denodo_odbc_driver

The Denodo Platform provides an ODBC driver for Linux, which is based on
the ODBC PostgreSQL driver.

As any other ODBC drivers, you have to install it in the machine where
the client application runs.
