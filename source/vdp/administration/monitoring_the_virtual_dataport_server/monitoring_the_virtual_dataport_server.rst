Monitoring the Virtual DataPort Server
======================================

The Denodo Platform provides two different ways of monitoring the
servers of the Platform: Virtual DataPort, Scheduler and ITPilot
browser pool and maintenance:

#. Using a Java Management Extensions (JMX) Agent such as JavaTM
   VisualVM. See section :ref:`Monitoring with a Java Management Extensions
   (JMX) Agent`.
#. Using the Denodo Monitor (application included in the Denodo
   Platform). See section :doc:`/vdp/administration/monitoring_the_virtual_dataport_server/denodo_monitor/denodo_monitor`.
