To begin using the VCS integration, you have to do the following:

#. Enable the VCS integration in the Server: see section :ref:`Virtual
   DataPort Server Configuration`.
#. Define at least one environment: see section :ref:`Environment
   Management`.
#. Configure an existing database to check it in a VCS server (see
   section :doc:`Database Configuration </vdp/administration/version_control_systems_integration/vcs_configuration/database_configuration>`) or import a Virtual DataPort
   database created by another user, from the VCS server (see section
   :ref:`Importing an Existing Database from a VCS Server`)
