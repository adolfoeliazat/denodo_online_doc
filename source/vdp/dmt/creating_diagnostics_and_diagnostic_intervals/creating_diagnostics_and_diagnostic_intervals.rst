=============================================
Creating Diagnostics and Diagnostic Intervals
=============================================

Before starting to diagnose an issue on a Virtual DataPort server,
obtain the diagnostic information of that Server and copy it into the
host where the Diagnostic & Monitoring Tool runs.

You can define diagnostic intervals to focus only on the information of
a certain period.
