=====================================================================
Migrating Wrappers Between Generation Environments: Import and Export
=====================================================================

The ITPilot generation environment provides a way to export the wrappers
to a file that can be imported into another generation environment. This
is useful for sharing the wrappers, recovering them after a clean
install or for backup purposes.
