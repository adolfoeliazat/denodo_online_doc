==============================================
Server Administration - Configuring the Server
==============================================

This section describes the main options available to administer the
Denodo Virtual DataPort server.

To manage the configuration a Virtual DataPort server, you have to log
in as an administrator user. When accessing the server for the first
time, use the default administrator credentials: login = “admin” and
password “admin”.

sections:
