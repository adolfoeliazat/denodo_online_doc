================================
Properties of the Navigation Bar
================================

By clicking on the Denodo icon on the left of the bar a dialog appears
which allows various generation process properties to be configured. The
dialog consists of three panels that allow configuring the preferences
for the criteria followed to generate NSEQL commands, the preferences
for authenticated proxies and the access method.
