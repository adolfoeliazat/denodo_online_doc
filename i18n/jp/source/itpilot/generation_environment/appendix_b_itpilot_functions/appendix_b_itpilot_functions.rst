This appendix describes the functions provided by ITPilot to create
expressions used in the Condition, Expression and Record Constructor
components.



ITPilot provides a series of predefined functions that can be grouped
into different types:

-  :ref:`Arithmetic functions <itp_gen_environment_guide_arithmetic_functions>`
-  :ref:`Functions for text processing <itp_gen_environment_guide_text_processing_functions>`
-  :ref:`Functions for handling null values <Functions for Handling Null Values>`
-  :ref:`Functions for handling lists <Functions for Handling Lists>`
-  :ref:`Functions for date processing <itp_gen_environment_guide_date_processing_functions>`
-  :ref:`Type conversion functions <itp_gen_environment_guide_type_conversion_functions>`
-  :ref:`Functions for URL processing <Functions for URL Processing>`
-  :ref:`Functions for page handling <Functions for Page Handling>`
-  :ref:`Extraction functions <Extraction Functions>`
-  :ref:`Functions for handling compressed files <Functions for Handling Compressed Files>`
-  :ref:`CASE clause <itp_gen_environment_guide_case_clause>`



The functions supported by the system are described in the following
paragraphs.



.. note:: Functions are generally represented in prefix notation, i.e.
   an identifier is indicated followed by a list of parameters delimited by
   parentheses and separated by commas:



Function\_name ( parameter1, parameter2, â€¦)



Remember also that starting in version 4.6 new functions can be added
through the use of user-created extensions (see :doc:`/itpilot/developer/index` for detail).
